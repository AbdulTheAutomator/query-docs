# File.Contents

## Syntax

<pre>
File.Contents(<b>path</b> as text, optional <b>options</b> as nullable record) as binary
</pre>
  
## About  

Returns the contents of the file, `path`, as binary. The `options` parameter is currently intended for internal use only.
