# Cube.AttributeMemberProperty

## Syntax

<pre>
Cube.AttributeMemberProperty(<b>attribute</b> as any, <b>propertyName</b> as text) as any
</pre>

## About
Returns the property `propertyName` of dimension attribute `attribute`.
