# ODataOmitValues.Type

## Definition

Specifies the kinds of values an OData service can omit.

## Allowed values

|Name|Value|Description|
| ------- | --- | ----------- |
|**ODataOmitValues.Nulls**|"nulls"|Allows the OData service to omit null values.|
